Extensão Bootcamp Helper

Esta é a página de apresentação da minha extensão, desenvolvida como parte do desafio do Bootcamp!

O que ela faz?

A extensão "Bootcamp Helper" é uma ferramenta simples que, ao ser ativada, permite ao usuário alterar a cor de fundo de qualquer página da web para um tom mais agradável, facilitando a leitura.

Como Usar

  1. Instale a extensão no seu Google Chrome no modo de desenvolvedor.
  2. Navegue para qualquer página da web.
  3. Clique no ícone da extensão na barra de ferramentas para abrir o popup.
  4. Clique no botão "Mudar Cor da Página".
  5. Pronto! O fundo da página será alterado.
